# 💪 GEM - Gym Exercise Manager

A modern and responsive Gym Exercise Manager built with React. The application allows users to log workouts, track training volume, monitor progress, and organize exercises by muscle groups through a clean and interactive interface.

---

## 🚀 Features

- 🏋️ Track workout sessions
- 📊 Daily training volume calculation
- 💪 Muscle group categorization
- ➕ Add new workout entries
- 🗑️ Remove logged exercises
- 🔥 Training streak section
- 🏆 Personal record highlight
- 📱 Responsive design
- ⚡ Fast performance with React + Vite

---

## 🛠️ Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS
- Lucide React Icons

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/gem-app.git
```

Go to the project folder

```bash
cd gem-app
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
